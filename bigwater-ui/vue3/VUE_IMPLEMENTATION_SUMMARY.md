## 🔄 工作流程

### 用户注册流程
1. 用户填写注册表单
2. 前端验证输入数据
3. 发送注册请求到后端
4. 后端创建用户并返回JWT令牌
5. 前端存储令牌和用户信息
6. 根据用户角色重定向到相应页面

### 用户登录流程
1. 用户输入邮箱和密码
2. 前端验证输入
3. 发送登录请求到后端
4. 后端验证凭据并返回JWT令牌
5. 前端存储令牌和用户信息
6. 重定向到用户仪表板

### 路由保护流程
1. 用户访问受保护的路由
2. 路由守卫检查认证状态
3. 如果未登录，重定向到登录页面
4. 如果已登录，检查用户权限
5. 根据权限决定是否允许访问
